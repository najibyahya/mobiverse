## Features
- Auto Mine
- Auto Activated Fever
- Auto Spin
- Single Akun ( tidak dibuat multi karna ini mining 24 jam )

## Get Hash
- Buka Game
- Buka Devtools/Inspect
- Refresh Game
- Menu Network
- Click "users"/"fever"/"count"/etc
- Headers
- Copy bagian user=xxxxxx / query_id=xxxxxxx
- Paste hash di token.txt
